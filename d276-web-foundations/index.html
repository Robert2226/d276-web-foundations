VLAN Lab Recap

Inside an OCI VM (Oracle Linux 9.6), I built a Linux bridge (br0) with VLAN filtering and two namespaces (ns-host, ns-gateway), connected via veth pairs. VLAN 10 tagging ensured pings only worked when ports matched the VLAN.

This exercise clarified how segmentation, tagging, and isolation map to real DC operations.
